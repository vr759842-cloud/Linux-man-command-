# Command Description :-

man :- Show manual pages of commands
man man :- Show manual page of man command itself
man ls :- Show manual page for ls command
man 8 ls :- Show section 8 (system admin commands) manual for ls (if available)
man -f ls :- Show short description of ls (same as whatis ls)
man intro :- Show introduction page of the manual system
man 1 intro :- Show section 1 (user commands) introduction
man 8 intro :- Show section 8 (system administration) introduction
whatis ls :- Show one-line description of ls command
man -k ls :- Search manual pages related to keyword ls (same as apropos ls)
man -w ls :- Show location of manual page file for ls
